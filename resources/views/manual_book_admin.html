Manual Admin Website

1. Dashboard

Halaman pertama setelah login. Menampilkan ringkasan statistik seperti jumlah produk, order, pelanggan, dan pengunjung.

2. Manajemen Admin

Menu ini untuk mengelola akun admin dan roles:

  • Daftar Admin → Menambahkan, mengedit, menonaktifkan akun admin.
  • Roles → Mengatur hak akses tiap admin.

3. Master Data

  • Kategori → Kelola kategori produk.
  • Produk → Tambah/edit produk. Gunakan ikon untuk edit, untuk hapus.
  • Banner → Kelola banner promosi.
  • Kupon → Tambah/edit kode kupon.
  • Bank → Kelola rekening pembayaran.

4. Manajemen Order

  • Pelanggan → Lihat daftar pelanggan.
  • Orderan → Pantau pesanan. Status aktif bisa menggunakan toggle /
  • Kalender Penggunaan → Melihat jadwal penggunaan aset/fasilitas.

5. Kerjasama

  • Kerjasama → Pantau kontrak/permintaan kerjasama.
  • Mitra → Tambah/edit mitra, gunakan toggle untuk status aktif/nonaktif, logo mitra akan muncul di tabel.

6. Dokumen dan Informasi

  • Informasi → Kelola artikel/informasi website.
  • Dokumen → Upload dokumen/formulir penting.

7. File Manager

  • Kelola semua file yang diupload ke server: produk, banner, dokumen, mitra.

8. System

  • Pengunjung → Statistik pengunjung.
  • Informasi & Log → Log aktivitas admin.
  • Database → Backup/restore database.

Tips Umum

  • Gunakan ikon toggle untuk mengganti status aktif/nonaktif: Aktif, Nonaktif.
  • Gunakan ikon edit dan hapus untuk operasi cepat.
  • Selalu periksa hak akses admin/staff sebelum memberi izin.
  • Backup database sebelum perubahan besar.
